# Embeddable Audio Player for Websites

A modern, lightweight audio player widget designed for seamless integration into websites. It features a customizable UI, playlist support, and a responsive audio visualizer powered by the Web Audio API. Built with performance and simplicity in mind, SoundBox allows developers to easily embed and control audio playback with minimal setup.

Launch Now: [https://deshan-nawanjana.github.io/soundbox/](https://deshan-nawanjana.github.io/soundbox/)

## Key Features

- Embed a fully functional audio player with a simple setup.
- Support multiple tracks with built-in playlist management.
- Responsive audio visualizer powered by the Web Audio API.

## Libraries and Frameworks Used

- [Vue.js](https://vuejs.org/) - Progressive framework for building user interfaces
- [PrismJS](https://prismjs.com/) - Syntax highlighting library
